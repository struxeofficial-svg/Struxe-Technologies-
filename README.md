# Struxe Technologies

A polished, responsive business website for Struxe Technologies. The site is built with plain
HTML, CSS, and JavaScript so it can be hosted easily on any static hosting provider.

## Files

- `index.html` - page content and structure
- `styles.css` - responsive visual design
- `script.js` - mobile navigation and contact form mailto behavior

## Preview locally

Open `index.html` in a browser, or serve the folder with:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
